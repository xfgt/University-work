Calculates the factorial of a number.\+This function computes the factorial of a given non-\/negative integer. It uses a recursive approach to calculate the factorial.


\begin{DoxyParams}{Аргументи}
{\em n} & The number for which the factorial is to be calculated.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Връща}
The factorial of the number.
\end{DoxyReturn}
The factorial of a non-\/negative integer n is the product of all positive integers less than or equal to n. For example, the factorial of 5 is 5 \texorpdfstring{$\ast$}{*} 4 \texorpdfstring{$\ast$}{*} 3 \texorpdfstring{$\ast$}{*} 2 \texorpdfstring{$\ast$}{*} 1 = 120. This function uses recursion to compute the factorial.

@usage int result = factorial(5); // result will be 120
